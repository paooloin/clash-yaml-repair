YAML Repair Tool

Clash YAML Repair Studio

修复乱码，还原 Unicode 转义字符，保持原始 YAML 结构。

支持上传本地 .yaml / .yml 文件，也支持输入 YAML 订阅链接进行即时修复。 当前版本不会保存你的配置，也不会生成公开链接。

上传 YAML 文件

选择本地 .yaml 或 .yml 文件，系统会自动修复乱码与转义字符。

订阅链接修复

输入返回 YAML 内容的订阅地址，由 Cloudflare Pages Function 拉取并修复。

当前版本仅支持 YAML 格式订阅，暂不解析 vmess / vless / trojan / ss 协议列表。

结构保留

不套用模板，不强行重组配置，只在原 YAML 基础上修复字符内容。

乱码修复

自动尝试修复常见 UTF-8 mojibake 乱码，让节点名称恢复可读。

Unicode 还原

将 \uXXXX 与 \UXXXXXXXX 转换为真实字符，并处理多余的引号转义。

快速导出

修复完成后可以直接复制结果，或下载为新的 YAML 文件。
